# 🐾 A.Pets

Uma aplicação web desenvolvida em Angular para conectar instituições de adoção de animais com futuros adotantes.

## 📑 Sobre o Projeto
O **A.Pets** busca facilitar o processo de adoção de animais que precisam de um lar. O foco do desenvolvimento foi aplicar conceitos de componentes reutilizáveis, serviços para gerenciamento de dados e uma interface amigável utilizando SCSS.

## 🛠️ Tecnologias e Ferramentas
* **Angular 19** (Framework principal)
* **TypeScript** (Lógica de programação)
* **SCSS** (Estilização avançada e variáveis)
* **Angular CLI** (Automação de tarefas)

## 🌟 Diferenciais do Projeto
- [x] Arquitetura modular e escalável.
- [x] Uso de pré-processador SCSS para um design consistente.
- [x] Interface intuitiva focada no usuário final.
